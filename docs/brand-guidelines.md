# ジモグル推し部 ブランディングガイドライン

## 1. ブランドアイデンティティ

### 1.1 ブランド定義
```
【ブランド名】
ジモグル推し部

【タグライン】
「地元の魅力をSNSで発信！ あなたもジモグル推し部で地域を盛り上げよう」

【ブランドストーリー】
地元の隠れた魅力を発見し、SNSを通じて広く発信することで、
地域の活性化に貢献する。グルメ好きとインフルエンサー志望者が
集まり、共に成長するコミュニティを目指す。
```

### 1.2 ミッション・ビジョン
```
【ミッション】
地元の魅力あるお店やグルメ情報をSNSで発信し、
地域経済と観光の活性化に貢献する

【ビジョン】
地元発のインフルエンサー育成を通じ、
地域全体の価値向上と新たなコミュニティ文化の創造を実現する

【バリュー】
- 地域愛：地元への深い愛着と貢献意欲
- 創造性：新しい価値の発見と発信
- 共創性：メンバー同士の学び合いと成長
```

## 2. ブランドパーソナリティ

### 2.1 性格設定
```
【メイン特性】
- フレンドリー：誰でも参加しやすい雰囲気
- パッション：グルメと情報発信への情熱
- プロフェッショナル：確かなスキルと知識

【コミュニケーションスタイル】
- 親しみやすい口調
- 具体的で分かりやすい説明
- 信頼感のある情報提供
```

### 2.2 ブランドボイス
```
【使用する言葉】
- 「一緒に」「みんなで」などの協調性を表す表現
- 「発見」「魅力」「可能性」などの前向きな言葉
- 「スキル」「ノウハウ」などの専門性を示す用語

【避ける表現】
- 過度に専門的な用語
- ネガティブな表現
- 過剰な誇張表現
```
